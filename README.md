# ServerSide_Crud

Title: Node.js + Express CRUD API with MongoDB

Description:
A simple CRUD (Create, Read, Update, Delete) REST API built using Node.js, Express, and MongoDB. The project follows a modular structure with separate folders for models, controllers, and routes. CORS and JSON parsing are enabled to allow seamless integration with frontend applications like Next.js.

Features:

Connects to MongoDB using Mongoose

Fully functional User CRUD API

Modular structure for scalability

Ready to integrate with frontend frameworks

Tech Stack:

Node.js

Express.js

MongoDB (Mongoose)

CORS
